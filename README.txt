Утилита для пробива по IP

Данная утилита поможет вам в ситуациях, когда надо пробить быстренько пробить по айпи. 

Установка

apt update -y && apt upgrade -y


apt install git python -y


pip install requests


git clone https://github.com/MrProCatYT/Uti-lite.git

Запуск

cd Uti-lite

python Uti-lite.py

Далее вас попросят вести айпи, и вуаля, в вашем кармане много ценной информации

Вот и все!
